# Smart-Traffic-and-Speed-detection-System
A multi-model computer vision system (uses colab T4 GPU). It trains three custom YOLOv11 object detection models from scratch on Roboflow datasets, then runs a unified frame-by-frame inference loop that simultaneously performs speed estimation, traffic-light surveillance, and licence-plate-based offender logging on any uploaded video
The smart surveillance and speed detection notebook has the pipeline for traffic light surveillance, license plate ocr post procesing and offenders logging, And Speed detection and calculation Pipeline.
